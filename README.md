# Flappy Bird - Mini

Phiên bản đơn giản của Flappy Bird bằng HTML/CSS/JS.

Tính năng:
- Điểm số
- Nút "Chơi lại"
- Va chạm với ống và đất/đỉnh
- Responsive cho điện thoại (canvas co dãn)

Chạy:
1. Mở `index.html` bằng trình duyệt (kéo vào trình duyệt hoặc dùng Live Server).
2. Chạm màn hình hoặc nhấn `Space` / click để nhảy.
3. Khi thua, nhấn nút "Chơi lại".

Files:
- `index.html` - trang chính
- `style.css` - style và layout
- `script.js` - logic game
